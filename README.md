# 🏥 医院排班助手

6天轮班制排班管理工具 — 门诊药房 → 住院药房 → 上午+夜班 → 休息×3

## 📱 安装到手机

### 方法一：GitHub Pages（推荐，免费）

1. 在 GitHub 新建仓库，上传这 4 个文件：
   - `index.html`、`manifest.json`、`sw.js`、`README.md`
2. Settings → Pages → Source 选 `main` 分支 → Save
3. 等几分钟，获得网址 `https://你的用户名.github.io/仓库名/`
4. **手机浏览器**打开这个网址
5. Chrome: 菜单 → **"添加到主屏幕"**
6. Safari: 分享 → **"添加到主屏幕"**
7. 📲 主屏幕上出现图标，点击就像 App！

### 方法二：国内托管（推荐备选）

- **Gitee Pages**（码云）：同上操作，国内访问更快
- **Cloudflare Pages**：全球加速

### 方法三：直接传文件到手机

1. 把 `index.html` 通过微信/QQ/AirDrop 发到手机
2. 用手机浏览器打开
3. 功能完全正常（但不能"安装"成独立 App）

## ⚙️ 首次设置

1. 点击右上角 ⚙️ 设置
2. 填写**轮班起始日期**（第1天=门诊药房的那天）
3. 保存即可

## 🔄 调班

- 点击日历上任一天 → 选择新班次 → 填写备注 → 保存
- 调过的日期会用**粉色虚线框**标出
- 下方表格可查看/删除调班记录

## 📅 日历提醒

点击 **"导出日历"** 下载 .ics 文件，导入手机系统日历，每天自动提醒！
